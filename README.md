# 🌍 Travelgram – Perfil de Viagens Responsivo

Este projeto é uma **página de perfil de viagens responsiva** desenvolvida com **HTML** e **CSS puro**, baseada em boas práticas de estrutura, semântica e organização de código.  
O objetivo é treinar **responsividade**, uso de **variáveis CSS** e **design moderno**.

---

## 🚀 Demonstração

🔗 [**Acesse o projeto online**](#) *(https://github.com/jessicaoliveiraaa/perfil_de_viagens)*  
---

## 📱 Funcionalidades

- Layout totalmente **responsivo** (mobile-first).  
- Navegação com elementos que aparecem ou desaparecem conforme o tamanho da tela (`mobile-only` e `desktop-only`).  
- Uso de **variáveis CSS** para cores, espaçamentos e fontes.  
- Estrutura organizada com **classes utilitárias** para facilitar manutenção.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** – Estrutura semântica do conteúdo.  
- **CSS3** – Responsividade, grid, flexbox, variáveis e reset.  
- **Google Fonts (Poppins)** – Tipografia moderna.
